Apa itu Air Quality Station?

Air Quality Station merupakan Project Mahasiswa Program Studi D4 - Teknik Rekayasa Elektronika yang berguna untuk mengukur kualitas udara. Project memanfaatkan berbagai macam sensor. Diantaranya adalah Sensor DHT22, Sensor MQ131, Sensor MQ136 dan Sensor CJMCU6814. Sensor DHT22 digunakan untuk pengukuran Suhu dan Kelembapan. Sensor MQ131 digunakan untuk pengukuran O3. Sensor MQ136 digunakan untuk pengukuran SO2. Dan Sensor CJMCU6814 digunakan untuk pengukuran CO, NH3, NO2.
